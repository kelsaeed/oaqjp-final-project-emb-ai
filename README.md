# Emotion Detector

An AI-based web application that detects emotions from text using the Watson NLP library.

## Project Description

This project uses IBM Watson NLP to analyze text and detect emotions such as anger, disgust, fear, joy, and sadness. It is deployed as a web application using Flask.

## How to Run

1. Install dependencies: `pip install flask requests`
2. Run the server: `python server.py`
3. Open browser: `http://localhost:5000`
